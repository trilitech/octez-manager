## Install in seconds

<Tabs>
  <TabItem label="Script">
```bash
curl -fsSL https://raw.githubusercontent.com/trilitech/octez-manager/main/install.sh | sh
```
  </TabItem>
  <TabItem label="From source">
```bash
git clone https://github.com/trilitech/octez-manager.git
cd octez-manager
opam install . --deps-only
dune build && dune install
```
See [full instructions](/getting-started/installation/#from-source) for OCaml/opam setup.
  </TabItem>
</Tabs>

## Deploy a node in one command

```bash
octez-manager install-node \
  --instance shadownet \
  --network shadownet \
  --history-mode rolling \
  --snapshot
```

Or launch the interactive TUI:

```bash
octez-manager
```

## Supported services

| Service | Command | Description |
|---------|---------|-------------|
| **Node** | `install-node` | Tezos L1 node with snapshot bootstrap |
| **DAL Node** | `install-dal-node` | Data Availability Layer |
| **Baker** | `install-baker` | Block production and attestations |
| **Accuser** | `install-accuser` | Double-baking detection |

## Manage instances

```bash
# List all instances
octez-manager list

# Control services
octez-manager instance shadownet start
octez-manager instance shadownet stop
octez-manager instance shadownet restart

# View logs
octez-manager instance shadownet logs

# Show details
octez-manager instance shadownet show
```
