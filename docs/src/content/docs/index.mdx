## Quick Start

```bash
# Install and launch the TUI
octez-manager

# Or use CLI directly
octez-manager install-node --instance my-node --network mainnet
```

## Supported Services

| Service | Description |
|---------|-------------|
| **Node** | Tezos L1 node |
| **Baker** | Block production and attestations |
| **Accuser** | Double-baking/endorsing detection |
| **DAL Node** | Data Availability Layer |
| **Signer** | Remote signing service |
